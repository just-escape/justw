{% include "svg/arrow-right.svg" %} Qu'est-ce qu'un escape game ?

Un escape game, ou jeu d'évasion, est un jeu d'équipe qui consiste à enfermer un groupe dans une escape room, une salle remplie d'énigmes. Le groupe de joueurs dispose de 60 minutes pour trouver les indices dissimulés dans la pièce.

Pour réussir la mission, les participants doivent faire preuve d'esprit d'équipe, de logique et d'un bon sens de la communication.

Le concept d'escape game est idéal pour se défier entre amis. Notre escape game futuriste n'attend que vous.

{% include "svg/arrow-right.svg" %} Pourquoi choisir un escape game pour votre EVJF ?

Un escape game pour votre EVJF ? Oui, mais pourquoi ?

  1. Expérience unique : un escape game suscite des émotions fortes, des rires et des souvenirs partagés.
  2. Renforcement des liens : les escape games nécessitent souvent une collaboration et une communication entre les membres du groupe pour résoudre les énigmes.
  3. Défi stimulant : les escape games peuvent être difficiles et stimulants, ce qui peut ajouter une dose d'excitation à votre EVJF et donner aux joueuses un sentiment d'accomplissement.
