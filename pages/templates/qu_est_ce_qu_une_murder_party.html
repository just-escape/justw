{% trans "Qu'est-ce qu'une murder party ? En bref" %}

{% trans "Une murder party, parfois appelé soirée enquête, soirée meurtre et mystère, murder diner, est une jeu d'immersion grandeur nature à huis clos dans lequel vous incarnez les personnages de l'histoire." %}

{% trans "Souvent, vous devez résoudre une intrigue policière, d'où le nom de “murder party”. Des comédiens vous accompagnent dans le scénario en jouant des personnages secondaires pour s'assurer que votre personnage progresse dans la bonne direction." %}

{% trans "Comment jouer à une murder party ?" %}

{% trans "Au début de la partie vous recevez des informations sur votre personnage :" %}

  • {% trans "Votre identité" %}
  • {% trans "Vos relations avec les autres : complicités, conflits, liaison secrète..." %}
  • {% trans "Vos objectifs, ce que vous devez accomplir pendant la soirée" %}

{% trans "Chaque participant incarne un personnage avec ses propres traits de caractères et ses propres objectifs. Vos objectifs peuvent être partiellement compatibles avec les autres personnages. C'est ce qui rend la murder party très fun. Il faudra donc faire des alliances, puis les trahir au bon moment !" %}

{% trans "On peut dire qu'une murder party est un mélange entre théâtre, improvisation, escape game, jeu d'enquête. On retrouve de plus en plus de mécanismes d'escape game intégrés à l'histoire, comme des coffres à ouvrir pour récupérer des indices !" %}

{% trans "Est-ce que je dois jouer comme sur une scène de théâtre ?" %}

{% trans "Il est recommandé aux joueurs de jouer, voire de surjouer leur personnage pour rendre les interactions amusantes." %}

{% trans "D'un autre côté, si vous vous sentez gêné à l'idée de devoir jouer théâtralement votre rôle au milieu de tout le monde : pas de panique. Vous pouvez passer toute la partie à comploter secrètement." %}

{% trans "L'important est que vos actions soient cohérentes avec votre personnage." %}

{% trans "Exemple : si vous jouez le rôle d'un banquier cupide, vous devez agir pour accumuler un maximum d'argent" %}

{% trans "Est-ce que je dois me déguiser ?" %}

{% trans "Les personnages reçoivent généralement des accessoires utilisables pendant la partie (sac à main, chapeau, vêtement supplémentaire). En fonction du thème, l'organisateur peut envoyer aux joueurs des suggestions vestimentaires pour améliorer l'immersion. Il arrive exceptionnellement que l'organisateur fournisse des costumes spécialement conçus pour la murder party (mais cela reste rare car cela pose des problèmes logistiques)." %}

{% trans "Encore une fois, rien n'est obligatoire. La plupart des murder parties se jouent avec des accessoires très légers." %}

{% trans "Combien de temps dure une murder party ?" %}

{% trans "Une murder party dure entre 2 et 3 heures. Certains scénarios sont pensés pour durer environ 1 heure et d’autres peuvent durer jusqu’à 10 heures, voire un week-end complet." %}

{% trans "Où se passe une murder party ?" %}

{% trans "Vous progressez en huis clos." %}

{% trans "Une murder party peut être organisée n'importe où. Dans ce cas il est nécessaire que le scénario soit bien travaillé pour éviter que les joueurs ne se dispersent." %}

{% trans "Les murder parties sont en général associées aux lieux insolites comme les châteaux ou les monuments historiques pour bénéficier des décors immersifs." %}

{% trans "Événements" %}

{% trans "Les événements sont une mécanique de jeu courante dans les murder parties. Il s'agit de rassembler tous les joueurs pour assister à une confrontation entre deux personnages, comme une joute verbale, un duel, une déclaration tragique. Le scénario peut prévoir des événements, connus à l'avance par les joueurs, ou au contraire qui vont arriver sans que personne ne soit au courant. Les joueurs peuvent aussi créer par eux-mêmes des événements." %}

{% trans "Exemple : Un jugement va avoir lieu dans une heure. Les personnages concernés par ce jugement vont devoir convaincre le juge de trancher en leur faveur." %}

{% trans "Plot twists" %}

{% trans "Dans un scénario de plusieurs heures, vous allez vivre des rebondissements. Les plot twists peuvent être soit prévus à l'origine dans le scénario, soit amenés par les joueurs." %}

{% trans "Exemple : Un personnage central de l'histoire meurt pendant la partie... Toutes les cartes sont rebattues. Si vous comptiez sur ce personnage pour réussir vos objectifs, il va falloir trouver d'autres solutions. Qu'allez-vous faire !?" %}

{% trans "Une murder party c'est comme un escape game ?" %}

{% trans "La murder party et l'escape game sont tous les deux des gens immersifs grandeur nature. Dans un escape game vous interagissez surtout avec des objets. Tandis que dans la murder party vous interagissez surtout avec les autres personnes." %}
